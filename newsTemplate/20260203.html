Today we are releasing GCompris version 26.0.

This new version also see the birth of the first official version of our teachers companion tool: gcompris-teachers, for which you can find more information on .

This version adds translation for two more languages: Kannada and Tamil.

This new version contains 197 activities, including 2 new ones:

  • "Drawing wheels" is an activity for drawing using a customizable gear in a cogwheel.
  • "Multiple choice questions" is a MCQ activity only visible when questionnaires have been sent by the teachers tool.

It contains bug fixes and improvements on multiple activities.
