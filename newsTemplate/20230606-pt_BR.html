Hoje estamos lançando a versão 3.3 do GCompris.

Esta versão adiciona traduções para mais 2 idiomas: árabe e esperanto.

Ela contém correções de bugs em várias atividades, tais como "Codificando o caminho", "Letra de uma palavra", "Apanhador de bola" e "Composição para piano".

Algumas melhorias no manuseio do teclado (atalhos, foco...) foram feitas em diversas atividades.

Também contém novas imagens e melhorias para "Investigador de fotos".

Ele foi totalmente traduzido para os seguintes idiomas:

    {% for oneLocale in fullyTranslatedLocales %}
  • {{ oneLocale }}
    • {% endfor %}

E também foi parcialmente traduzido para os seguintes idiomas:

    {% for oneLocale,oneStatus in partiallyTranslatedLocales %}
  • {{ oneLocale }} ({{ oneStatus }}%)
    • {% endfor %}

Você pode encontrar os pacotes desta nova versão para GNU/Linux, Windows, Raspberry Pi e MacOS na página de downloads. Esta atualização também estará disponível em breve na Play Store do Android, no repositório F-Droid e na Windows Store.
