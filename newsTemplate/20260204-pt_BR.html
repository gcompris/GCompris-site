Hoje estamos lançando a versão 26.0 do GCompris.

Também estamos lançando a primeira versão oficial de nossa ferramenta complementar para professores: GCompris-teachers! Você pode encontrar mais informações sobre ela na página sobre escolas.

Esta nova versão contém 197 atividades, incluindo 2 novas:

  • "Desenhar rodas" é uma atividade para desenhar usando uma engrenagem girando em uma roda dentada.
  • "Questões de múltipla escolha" é uma atividade de múltipla escolha. Observe que esta atividade está oculta por padrão. Ela se torna visível depois que alguns conjuntos de dados para ela forem enviados pelos professores do GCompris.

Ela contém correções de bugs e melhorias em várias atividades.

Enviamos traduções para mais dois idiomas: canarês e tâmil.

Você encontra os pacotes desta nova versão para GNU/Linux, Windows, Android e Raspberry Pi na página de downloads. Esta atualização também estará disponível, em breve, na Play Store do Android, no repositório F-Droid e na Windows Store.
