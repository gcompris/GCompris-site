Today we are releasing GCompris version 4.0.

This version adds translations for 3 more languages: Bulgarian, Galician and Kiswahili.

It contains 190 activities, including 8 new ones:

  • "Grammar analysis" is an activity for children to identify grammatical classes in the given sentences.
  • In "Grammar classes" is similar to the previous one but the children here focus on one specific grammatical class.
  • "Calcudoku" is an arithmetic game where we need to fill a grid with numbers according to specific rules.
  • With "Guess 24", using the 4 numbers and the operators, find the number 24!
  • In "Frieze", reproduce and complete the different friezes.
  • "Read a graduated line" is an activity when you need to find the value represented in a line.
  • In "Use a graduated line", find where in the line is the given value.
  • In "Adjacent numbers", learn which numbers come before of after the given sequence.

It contains bug fixes and graphics improvements on multiple activities.

One major milestone has been reached with this version: after 8 years of work, all the graphics have been reworked to fit the guidelines!
